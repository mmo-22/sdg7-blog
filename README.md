# SDG 7 Project Website — EN8905 (Bahrain Polytechnic)

Static website for the Individual Project, "Sustainable Energy Systems & the SDGs".
8 entries covering all 5 required stages. Regional focus: Bahrain. No build step.

## Deploy

GitHub Pages (recommended — must stay live 15–20 July for viva):
1. Public repo → push files → Settings → Pages → Source: main / root
2. Test the link in an incognito window before pasting it into Slide 2

If staying on Railway: make sure the service cannot sleep or expire before 20 July.

## Workflow

- Amber TODO boxes contain instructions mapped to the rubric — write your
  content, then DELETE every .todo box before publishing.
- Replace all [ ... ] placeholders; state the source + year for EVERY number.
- Evidence photos go in assets/ (IMG-meter-photo.jpg, IMG-session-photo.jpg,
  IMG-demand-chart.png) or update the <img src> paths.
- DO NOT re-run generate.py after writing content — it overwrites all pages.
